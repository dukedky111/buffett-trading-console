# 交易策略总纲

## 1. 策略定位

策略名称：`Buffett Transcript Value Signal`

本策略不是短线喊单系统，而是一套价值投资筛选与交易提醒系统。

YouTube transcript 的作用是发现股票线索，不能直接触发买入。真正决定交易的是：

1. 是否在能力圈内。
2. 是否是好生意。
3. 是否有可持续护城河。
4. 管理层是否诚实且理性。
5. 财务质量是否真实可靠。
6. 当前价格是否明显低于保守内在价值。
7. 仓位风险是否可控。

## 2. Buffett 八问过滤

每只候选股票先回答八个问题：

| 编号 | 问题 | 否定含义 |
| --- | --- | --- |
| 1 | 我能否用一段话讲清楚它怎么赚钱？ | 不在能力圈 |
| 2 | 这家公司 10 年后还会存在，并且更强吗？ | 长期确定性不足 |
| 3 | 它有没有别人很难复制的护城河？ | 缺少长期超额收益来源 |
| 4 | 它能不能涨价 5%-10% 而不大量流失客户？ | 缺少定价权 |
| 5 | 利润能不能真实转化成现金流？ | 盈利质量差 |
| 6 | 收入下跌 30% 时，它还能活下来吗？ | 资产负债表风险高 |
| 7 | 管理层是否诚实、理性、股东导向？ | 一票否决 |
| 8 | 当前价格是否明显低于内在价值？ | 只能等待 |

规则：

- 1-2 个 No：继续研究，但必须写出强理由。
- 3 个 No：只能观察，不能买入。
- 4 个以上 No：放弃。
- 管理层诚信问题一票否决。
- 价格不便宜时，只能等待。

## 3. 企业质量评分

满分 100。

| 维度 | 分数 |
| --- | --- |
| 能力圈 | 10 |
| 商业模式清晰 | 10 |
| 护城河强度 | 20 |
| 定价权 | 10 |
| ROIC / ROE 质量 | 15 |
| 现金流质量 | 15 |
| 资产负债表安全 | 10 |
| 管理层与资本配置 | 10 |

等级：

| 分数 | 解释 |
| --- | --- |
| 85-100 | 核心候选，可以深入研究和等待重仓机会 |
| 75-84 | 优秀候选，可以等待好价格 |
| 65-74 | 普通候选，只允许小仓位 |
| <65 | 不交易 |

## 4. 护城河判断

优先选择以下类型：

- 品牌护城河：客户愿意为品牌付溢价。
- 低成本护城河：成本结构长期优于竞争对手。
- 转换成本：客户更换供应商的成本高。
- 网络效应：用户越多，产品越有价值。
- 有效规模：市场天然只能容纳少数玩家。

关键问题：

```text
如果公司涨价 10%，客户会不会离开？
竞争对手花 5 年和大量资本能不能复制它？
毛利率、市场份额、客户留存率是在变强还是变弱？
```

## 5. 财务质量

核心指标不是 EPS，而是 owner earnings。

```text
Owner Earnings =
净利润
+ 折旧摊销
- 维持竞争地位所需资本开支
- 必要营运资本增加
```

主要阈值：

| 指标 | 理想状态 |
| --- | --- |
| 10 年平均 ROE | > 15%，且不是靠高杠杆 |
| ROIC | 长期 > 15%，且高于 WACC |
| 经营现金流 / 净利润 | > 90% |
| Debt / EBITDA | < 2 |
| EBIT / Interest | > 5 |

红旗：

- 应收账款长期快于收入增长。
- 库存长期快于收入增长。
- 经营现金流长期低于净利润。
- 频繁使用 adjusted earnings 掩盖真实问题。
- 大额商誉来自高价收购，但回报平庸。
- 关联交易频繁且不透明。
- 审计师频繁更换。
- 管理层大量卖股。

## 6. 估值与安全边际

估值使用三种方法交叉验证：

1. Owner Earnings × 合理倍数。
2. 简化 DCF。
3. 7-10 年平均盈利能力 × 保守倍数。

倍数参考：

| 公司类型 | 合理倍数 |
| --- | --- |
| 宽护城河 + 轻资产 + 稳定增长 | 20-25x |
| 中等护城河 + 稳定增长 | 15-20x |
| 普通公司 | 10-15x |
| 周期、衰退或高度不确定公司 | 8-12x 或放弃 |

安全边际：

| 公司确定性 | 最低安全边际 |
| --- | --- |
| 极优秀公司 | 20%-30% |
| 一般优秀公司 | 30%-40% |
| 不确定公司 | 40%-50% |
| 无法估值 | 不投资 |

默认买入硬条件：

```text
当前价格 <= 保守内在价值 × 70%
```

极高质量且高确定性的公司，可放宽到：

```text
当前价格 <= 保守内在价值 × 80%
```

## 7. 买入规则

必须同时满足：

```text
Buffett Quality Score >= 75
安全边际 >= 30%
无管理层诚信问题
无重大财务红旗
视频信号评分 >= 60，或股票已经在自选池
价格接近估值买入区或长期支撑位
组合风险敞口允许
```

买入动作：

| 动作 | 条件 |
| --- | --- |
| 观察 | 质量好，但价格贵 |
| 试仓 | 质量好，价格接近合理，安全边际刚达标 |
| 正式建仓 | 质量好，价格明显便宜，视频/点位/基本面一致 |

## 8. 加仓规则

允许加仓：

- 原始买入逻辑仍成立。
- 财务和护城河没有恶化。
- 价格比上次买入低 10%-15%。
- 安全边际扩大。
- 单票仓位没有超过上限。

禁止加仓：

- 只是为了摊低成本。
- 公司基本面恶化。
- 现金流质量变差。
- 负债风险上升。
- 护城河被技术或竞争破坏。

## 9. 卖出规则

卖出只有四类理由：

1. 价格严重高于内在价值。
2. 护城河被结构性破坏。
3. 管理层诚信出现问题。
4. 出现明显更好的机会。

估值卖出规则：

```text
价格 >= 保守内在价值 × 1.1：考虑减仓
价格 >= 保守内在价值 × 1.3：考虑大幅减仓
价格 >= 保守内在价值 × 1.5：除非超级复利资产，否则卖出
```

立即卖出：

- 财务造假。
- 管理层欺骗。
- 核心业务被永久性替代。
- 现金流质量崩坏。
- 债务危机导致稀释融资风险。

不该卖出的理由：

- 股价跌了。
- 大盘跌了。
- 分析师降级了。
- 视频变谨慎了。
- 短期财报低于预期，但长期逻辑没变。

## 10. 仓位管理

组合结构：

| 仓位层 | 占比 |
| --- | --- |
| 核心价值仓 | 60%-70% |
| 机会观察仓 | 20%-30% |
| 现金 | 10%-20% |

单票仓位：

| 类型 | 仓位 |
| --- | --- |
| 普通优秀公司 | 2%-4% |
| 高确定性优秀公司 | 5%-8% |
| 极高确定性核心公司 | 10%-12% |
| 不确定但有机会 | <= 2% |

限制：

- 单一行业不超过 30%。
- 高相关科技成长股合计不超过 35%。
- 周期股合计不超过 20%。
- 高杠杆公司合计不超过 10%。
- 不使用融资。

## 11. 最终哲学

```text
市场预测：只听，不交易。
视频观点：只做线索，不做命令。
点位：只做执行，不做价值判断。
估值：决定能不能买。
护城河：决定能不能长期持有。
管理层诚信：一票否决。
安全边际：决定风险是否值得承担。
仓位：由确定性决定。
卖出：由价值和企业质量决定，不由恐慌决定。
```
